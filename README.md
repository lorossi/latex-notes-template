# Latex Notes Template

Since I decided to take all my university notes in Latex, I decided to stop copying files between projects and folder and start making a common template.
This package includes all the LaTeX packages that I need and a default empty document.

Included packages:

- `geometry`
- `inputenc`
- `ccicons`
- `xcolor`
- `csquotes`
- `babel`
- `fontenc`
- `lmodern`
- `tikz`
- `amsmath`
- `amsfonts`
- `amssymb`
- `mathtools`
- `wasysym`
- `hyperref`
- `verbatim`
- `listings`
- `enumitem`
- `graphicx`
- `fontawesome`
- `titlesec`
- `amsmath`
- `xspace`
- `xfrac`
- `tocloft`
- `nameref`
- `courier`
- `anyfontsize`
- `subcaption`
- `datetime2`
